GirlScript Jaipur Induction Meetup
Sat, Feb 2, 2019 10:00 AM

Overview

Free and Open Source Software

Sarvesh Jhanwar

In this session, you'll get to know about Open Source Software and its Development, along with its importance. We'll guide you through the process of contribution and several programs associated with the process of becoming a regular contributor.Furthermore, you'll learn about Version Control Systems, and discuss Git and GitHub in detail. This will be followed by a hands-on GitHub session.

Web Development

Arpit Goyal

In this session, you'll get to know about Full stack web development. The Technologies behind Front-end and Back-end Web development like HTML, CSS, JavaScript etc. You will get to learn about what all you can do with JavaScript like making Chrome extensions, building servers and there will be a hands-on training on how to build your own Chrome extension using JavaScript.

Selecting your first Programming Language

Shubhangi Verma

The session will include basics of C, C++ and Object Oriented Programming along with the overview of languages like java and python as well as the present needs of the industry and the scope of these languages in the future. Students will get help in selecting their first programming language based on the information given and their interests. The session will act as a visual guide which will showcase the pros and cons of all the major languages so that students can make an informed decision.

Event Recap

"Talent wins games, but teamwork and intelligence win championships."
--Michael Jordan

GirlScript Foundation started it's new chapter in Jaipur with a hope of catering the needs of the industry by providing knowledge about what exactly is needed for the young minds to be industry ready. For this thought to get a kickstart it was necessary to come together and know each other better. To fulfill this purpose we planned a meetup for the beginners in technology to come up and learn something about us. The meetup was a great success, it had people who were actually eager to learn and know who we are and what we wanted to do.
